IMPORTANTE!

read(): lê o arquivo totalmente a partir do cursor
readline: lê somente a linha onde esta o cursor
readlines: transforma cada linha do arquivo em um elemento de uma lista de strings 
seek: posiciona o cursor em algum ponto do arquivo txt
len: conta quantos elementos há em uma lista ou conta quantos charactéres há em uma string
split(): transforma string em lista
